CSS İpuçları 5 : CSS seçicilerini tanımlarken küçük-büyük harfe dikkat etmek

Thu 20 July 2006
By Fatih

In CSS.

tags: CSShtmlküçük büyük harfseçiciselector

Eğer XHTML ile CSS kullanıyorsak seçici(selector) olarak kullandığımız XHTML elementlerinin küçük büyük harfe duyarlı olduğunu unutmamalıyız. Küçük büyük harf farklılıklarından kaynaklanan sorunları aşmak için XHTML elementi olarak kullandığımız seçicileri küçük harfle yazarak bu sorunu bertaraf etmiş oluruz. HTML ve XHTML'in her ikisinde de özellikler(attributes) küçük büyük harfe duyarlıdır. Eğer farklı harf büyüklükleri kullanıyorsanız buna çok fazla dikkat etmelisiniz. [sourcecode language="css"] .DenemeYazisi{ font:12px Arial, Verdan, serif } [/sourcecode] [sourcecode language="html"] \<p class="DenemeYazisi">Dikkatli ol!\</p> [/sourcecode]
