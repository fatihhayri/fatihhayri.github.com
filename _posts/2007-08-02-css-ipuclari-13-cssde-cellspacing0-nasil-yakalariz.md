Artık sayfalarımızı CSS ile oluşturup şekillendiriyoruz. Tabloların
Cellspacing="0" atamasını css ile nasıl yaparız. İşte cevabı:

	:::css
	table{
		border-spacing: 0px;
		border-collapse: collapse;
	}


bu kadar.
