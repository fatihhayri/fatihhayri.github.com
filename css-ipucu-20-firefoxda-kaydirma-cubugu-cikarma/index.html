CSS İpucu 20: Firefox'da Kaydırma Çubuğu Çıkarma

Thu 16 April 2009
By Fatih

In CSS.

tags: Firefoxipucukaydırma-çubuğumozillaoverflow

Firefox İnternet Explorer'un tersine eğer sayfa yüksekliği kaydırma çubuğu çıkarmayacak kadar yüksekliğe sahip değilse kaydırma çubuğu çıkarmıyor. Bu özellikle çerçeveli sayfalarda sorun çıkarabilir. Bunun için bir çözüm. [sourcecode language="css"] html{ overflow:-moz-scrollbars-vertical; } [/sourcecode]
